# Contributing

感谢你愿意帮助 Tony Na Engine。

## 适合贡献的方向

- Bug 修复
- UI / 交互打磨
- 剧情编辑器体验优化
- 导出链与打包链改进
- 自动化测试补充
- 文档与新手说明

## 提交前建议

1. 先阅读仓库里的说明文档
2. 优先在 Issue 里说明你准备改什么
3. 尽量保持改动聚焦，不要一口气混很多不相关修改
4. 如果改到功能逻辑，请尽量补测试

## 本地运行

```bash
cd "/Users/na/Game Engine"
python3 run_editor.py
```

## 测试

```bash
cd "/Users/na/Game Engine"
python3 -m unittest discover -s tests -v
```

## 提交风格建议

- 路径和命名尽量清楚
- 优先修真实问题，不做空泛重构
- 如果是 UI 修改，尽量说明影响范围
- 如果是导出链或打包链修改，尽量说明在哪个平台验证过

## 需要特别注意

- 不要提交真实证书、密码、私钥、PFX
- 不要把临时导出物、缓存、系统文件一起提交
- 不要随便改示例项目里已经用于自动化测试的关键素材和流程，除非你同步修了测试
