# Security Policy

## Supported Versions

当前仓库仍处于 `Source-Available Preview` 阶段，默认只维护主分支上的最新代码与最近一次公开预览版。

## Reporting a Vulnerability

如果发现的是**安全问题**，建议不要直接公开提交 Issue。

请优先通过以下方式私下联系：

- 在仓库资料页提供的支持邮箱联系维护者
- 或在尚未配置支持邮箱前，先通过 GitHub 私信 / 私下渠道联系仓库维护者

报告时建议附上：

- 影响范围
- 复现步骤
- 可能的利用方式
- 临时缓解方案（如果有）

在问题确认前，建议不要公开披露细节。
