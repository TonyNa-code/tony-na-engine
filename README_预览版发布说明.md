# Tony Na Engine 预览版发布说明

在还没有商业签名证书之前，Tony Na Engine 仍然可以先以 **预览版 / 内测版 / 开源创作者预览版** 的形式发布。

## 最适合现在阶段的定位

建议在 GitHub Releases 或仓库首页明确标注：

- `Preview`
- `Early Access`
- `Experimental`
- `Unsigned Build`

## 推荐分发方式

优先发布：

- `三系统编辑器套装`
- 或单独平台编辑器桌面包

## 需要提前说明的事项

### macOS

- 第一次打开可能会提示“未知开发者”
- 用户需要手动放行

### Windows

- 可能会遇到 SmartScreen 警告
- 用户需要手动选择继续运行

## 发布页建议明确说明

1. 这是一版预览版，不是最终稳定版
2. 现在重点是收集创作者反馈
3. 如果遇到问题，请提 Issue
4. 目前还没有完成商业签名

## 推荐发布内容

建议优先发布：

- 源码仓库
- 一份 GitHub Release
- 一套编辑器压缩包
- 一份简短的首次打开说明

这样用户即使第一次被系统拦住，也能明确知道这是预期内行为，而不是软件损坏。
