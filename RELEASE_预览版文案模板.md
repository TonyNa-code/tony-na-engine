# GitHub Release 文案模板（预览版）

## Tony Na Engine Preview

Tony Na Engine 是一套面向视觉小说 / galgame 创作者的可视化引擎原型。

这一版是：

- 开源创作者预览版
- 适合提前试做项目
- 适合反馈编辑器体验、剧情工作流、导出链、配音工作流和粒子系统

### 这版包含

- 可视化剧情编辑器
- 项目中心与空白项目
- 新手模式 / 高级模式
- 正式存档 / 读档
- 各类回想馆 / 图鉴馆 / 成就馆
- 配音工作流
- 高级粒子系统
- 网页试玩包 / Windows 桌面包 / 编辑器桌面包

### 需要注意

- 这还是预览版，不是最终稳定版
- 当前分发包可能未做商业签名
- macOS / Windows 首次运行可能会出现系统安全提示

### 如果你要反馈问题

欢迎直接提 Issue，最好附上：

- 你的系统版本
- 复现步骤
- 报错截图
- 使用的是哪个导出包或编辑器包
